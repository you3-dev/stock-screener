# 10 段C レジーム/リスクオーバーレイ — 単純トレンドフィルタは強くGO

> 統合設計 docs/06 §6・層C。**当てる(方向予測)のではなく、地合い悪化時に新規スイングを控える/外す**ことで、
> 任意ロング母集団の最大ドローダウンが縮むか(=負けにくさ)を検証。`src/overlay/regime.py` /
> `scripts/run_regime_overlay.py`。実施日: 2026-06-24。**§1「マクロを選定シグナルに使わない」を遵守
> (銘柄選別でなくポートフォリオ単位のリスクゲート)。**

## 方法
- マクロ: `^N225` / `^GSPC` 日足(yfinance、5年)。**2市場は立会日が違うため系列ごとにNaN除去してから**
  50日線・60日ローリング相関を算出(混在NaNでローリングが全滅するバグを修正済)。
- レジーム: **risk_off = GSPC < 50日線**、high_corr = 60日相関 ≥ 閾値。ルックアヘッド防止に `shift(1)`
  (t-1までの情報でt寄りに行動可能)。
- 母集団 = ユニバースの**日次クロスセクション等加重平均リターン**(=分散ロングが実際に得る値)。All と Small(Std+Growth)。
- 比較: ベースライン(常時保有) vs オーバーレイ(ゲート日は現金=新規・保有を外す強形)。
  指標 = 累積リターン・最大DD・リターン/|最大DD|・在市場率。
- **複数定義を併記**(チェリーピック回避)。

## 結果

| 母集団 | 定義 | ベース ret / DD / r-DD | オーバーレイ ret / DD / r-DD | 在市場 | 判定 |
|---|---|---|---|---|---|
| 全ユニバース | **risk-off (GSPC<50d)** | +75.5% / -21.2% / 3.55 | **+140.1% / -6.9% / 20.3** | 69% | **GO** |
| 全ユニバース | corr≥0.5 & risk-off | 〃 | +52.2% / -26.2% / 1.99 | ~100% | no |
| 全ユニバース | corr≥0.4 & risk-off | 〃 | +57.0% / -24.2% / 2.36 | 99% | no |
| Small | **risk-off (GSPC<50d)** | +49.6% / -21.7% / 2.29 | **+106.3% / -7.5% / 14.1** | 69% | **GO** |
| Small | corr≥0.5 & risk-off | 〃 | +29.0% / -26.4% / 1.10 | ~100% | no |

### 解釈
1. **単純な risk-off フィルタ(GSPC<50日線で新規を外す)が強く効く**: 最大DDを **約-21% → -7%** に圧縮し、
   **リターンはむしろ増加**(全 +75%→+140% / 小型 +50%→+106%)。リスクオフ日の平均リターンが負のため、
   避けるとDDもリターンも改善 = **単なるβ縮小ではない**(r/DDが3.55→20.3 と大幅改善)。在市場は69%(約31%は様子見)。
2. **「高相関」の絞りはむしろ悪化**。high_corr を要求するとゲートが弱まり(ほぼ常時在市場)、かつ
   高相関日に含まれる**急反発(ベア相場のリバウンド)を取り逃す**ため逆効果。→ **当初仮説の"相関"要素は棄却。
   プレーンなトレンドフィルタが本命。**

## 留意・最弱の前提(過信しないために)
- **これはトレンドフォロー(時系列モメンタム)**。SPX下降局面を避ける古典的手法で、効果は確立しているが、
  **5年の便益の大半は2022年の世界同時安1エピソードに依存**(finance §6.4「特定期間に依存しすぎない」を
  完全には満たさない)。+140%という大きさは過大評価。**「DDを構造的に縮める」点は信頼、倍率は割り引く。**
- **強形(ゲート日に全現金)は理想化**。実運用は「**新規スイングを控える/サイズを縮める**」が現実的で、効果は
  この上限より小さい。スイッチ頻度は低く(レジームは持続)取引コストは軽微。
- **日本株に米国トレンド(GSPC)を当てる**: 世界同時リスクオフでは有効だが、JP固有の下げ(米が平穏)は取り逃す。
  N225版・併用版は今後の調整余地。

## 判定: **GO(採用=層C)**
- 「GSPC<50日線のリスクオフ時に新規スイングを控える/サイズを縮める」は、All・Small 両母集団で**最大DDを
  約3分の1に圧縮**し、β縮小では説明できない改善を示した。**§1遵守(方向当てでなくリスクゲート)**。
- ただし**2022依存・トレンドフォローの一般的限界**を明記して使う。倍率でなく**DD抑制効果**を信頼する。
