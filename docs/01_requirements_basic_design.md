# 株価期待値スクリーニングツール

## 要件定義書・基本設計書 v1.0

作成日: 2026-02-21

------------------------------------------------------------------------

# 1. システム概要

## 1.1 目的

東証プライム全銘柄を対象に、翌日寄り付きエントリー時点で統計的に期待値が正となる銘柄のみを抽出する投資判断支援Webアプリを構築する。

本ツールは自動売買は行わず、人間による最終判断を前提とする。

------------------------------------------------------------------------

# 2. 投資戦略前提条件

-   対象市場: 東証プライム
-   現物ロングのみ（信用取引なし）
-   エントリー: 寄り付き
-   売却: 翌日以降寄り付き
-   最大保有期間: 20営業日（強制売却）
-   同時保有最大: 5銘柄
-   等金額配分
-   手数料・税金は考慮しない
-   ナンピンなし

------------------------------------------------------------------------

# 3. データ仕様

## 3.1 分析期間

過去5年間

## 3.2 重み付け方式

指数減衰型重み付け

weight = exp(-λ × 経過年数)

λは0.3を初期値とする。

------------------------------------------------------------------------

# 4. 前日夜スクリーニング条件

## 4.1 流動性フィルター

-   前日出来高 ≥ 500,000株（検証用下限 100,000株）
-   売買代金 ≥ 5億円
-   株価 800〜6000円

## 4.2 資金流入検知条件

-   売買代金5日平均比 ≥ 1.8倍
-   20日高値更新
-   前日陽線
-   ATR(14) / 株価 ≥ 2%
-   直近3日騰落率 ≤ +12%

## 4.3 除外条件

-   ストップ高翌日銘柄
-   前日終値比 +5%以上ギャップ想定銘柄

------------------------------------------------------------------------

# 5. 期待値計算ロジック

## 5.1 リターン定義

R(n) = n営業日後寄り価格 / エントリー寄り価格 - 1\
n = 1〜20

## 5.2 出力統計

-   重み付き中央値リターン
-   重み付き勝率
-   重み付き最大DD中央値

## 5.3 エントリー基準（初期値）

-   1日期待値 ≥ +0.35%
-   勝率 ≥ 57%
-   DD中央値 ≥ -1.8%以内

## 5.4 推奨保有日数

n=1〜20の中で期待値最大の日数を採用する。\
DDが-3%超の場合は警告表示。

------------------------------------------------------------------------

# 6. 当日寄り気配チェック
